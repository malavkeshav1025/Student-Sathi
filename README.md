# 🎓 Student Sathi

Student Sathi is a student support platform designed to help students share their academic and personal challenges and receive guidance in a safe and supportive environment.

## 🚀 Features

* Student-friendly interface
* Share academic concerns
* Share personal and mental well-being related concerns
* One-to-one guidance support
* Easy communication through social platforms
* Responsive design

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

## 🎯 Project Objective

The objective of Student Sathi is to provide students with a platform where they can express their concerns and seek guidance without hesitation.

## 💡 Key Highlights

* Focused on student well-being
* Simple and user-friendly design
* Encourages communication and support
* Built as a social impact project
